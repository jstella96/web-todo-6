## Quick Start
### 1. Clone & Install Packages
```
git clone https://github.com/woowa-techcamp-2022/web-todo-6.git

cd client
npm install

cd ../server
npm install
```

### 2. Setup Environment Variables
```bash
# server에 .env 파일 생성

DB_URL= 
DB_ID= 
DB_PASSWORD=
DB_DATABASE=
DB_PORT=
```

### 3. Run Application
```bash
# ./client 
npm run dev

# ./server 
npm run nodemon
```
